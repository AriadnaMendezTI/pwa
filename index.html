Proyecto PWA

Creación de una aplicación web progresiva

Realizado por Jazmin Mendez y Giovana Soledad

Una aplicación web progresiva (PWA por sus siglas en inglés) es un tipo de software de aplicación que se entrega a través de la web, creado utilizando tecnologías web comunes como HTML, CSS y JavaScript. Está destinado a funcionar en cualquier plataforma que use un navegador compatible con los estándares.
